Synopsis

Removes the node from its current parent (decrementing its descendant count) and adds it as a descendant of the supplied new_parent node. If this removes a numlist node's last descendant, the numlist node is automatically deleted.

Return Value

False if no change is made. This happens if new_parent is already a descendant of the current node or the node's current parent.

Arguments

new_parent [jpx_metanode]
