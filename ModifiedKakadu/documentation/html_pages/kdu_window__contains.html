Synopsis

Returns true if rhs describes a window which is fully contained within the window described by the current object. Containment refers not only to the image window geometry, image components, image resolution, image region and code-streams, but also to the number of quality layers and the collection of metadata elements which are of interest, as expressed by the various members of the present structure. Does not check expansions (see kdu_sampled_range::expansion), since the expansion for each range in the contexts set should be unique.
