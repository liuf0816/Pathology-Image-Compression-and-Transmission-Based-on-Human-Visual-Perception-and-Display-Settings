Synopsis

Returns the total number of bytes which have been received from the server, including all headers and other overhead information. This function differs from kdu_cache::get_transferred_bytes in two respects: firstly, it returns only the amount of data transferred by the server, not including additional bytes which may have been retrieved from a local cache; secondly, it includes all the transfer overhead.
