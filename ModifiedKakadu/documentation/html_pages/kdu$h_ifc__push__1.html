Synopsis

Delivers all samples from the line buffer across the interface.

Arguments

line [kdu_line_buf &]

env [kdu_thread_env *]

If the object was constructed for multi-threaded processing (see the constructors for kdu_analysis and kdu_encoder), you MUST pass a non-NULL env argument in here, identifying the thread which is performing the push call. Otherwise, the env argument should be ignored.
