Synopsis

Use this function to change the type of the current node to one which represents a label ( lbl ) box, if necessary, and replace its contents (if any) with the supplied label string. If the node already has descendants, it is represented by an association box, whose first sub-box is now a label box.

Arguments

text [const char *]

Null-terminated text string to be stored in the body of the label box; the string is copied internally.
