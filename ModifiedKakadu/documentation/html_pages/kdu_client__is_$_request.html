Synopsis

Returns true if the request string passed to connect represented a one-time request, as defined there. In particular, a one-time request is one which contains a query string with anything other than a target or sub-target request field. In the event that a one-time request was found in the request string passed to connect, this function continues to return true until close is called, i.e., for the same interval over which is_active returns true.
