jpx_metanode::get_regions

const jpx_roi * get_regions( )

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Returns a pointer to the internal array of regions (and their attributes) associatd with an ROI description box.

Return Value

Non-NULL only if the node represents an ROI description box; i.e., if get_num_regions returns non-zero.
