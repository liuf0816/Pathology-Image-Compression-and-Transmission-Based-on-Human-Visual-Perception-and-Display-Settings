jpx_metanode::change_to_delayed

void change_to_delayed( kdu_uint32 box_type, int i_param, void * addr_param)

[Declared in "../apps/compressed_io/jpx.h"]

Go to class description.

Synopsis

Use this function to change the type and/or delayed node identification parameters of the current node. The operation is similar to deleting the existing node and then reinstantiating it using add_delayed, except that the descendants and order within its sibling list are retained.
