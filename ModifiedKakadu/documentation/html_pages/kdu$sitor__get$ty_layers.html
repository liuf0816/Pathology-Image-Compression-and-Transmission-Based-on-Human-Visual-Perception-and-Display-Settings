Synopsis

Returns the maximum number of quality layers which are available (even if not yet loaded into a dynamic cache) from any code-stream currently in use. This value is useful when determining a suitable value to be supplied to set_max_quality_layers. Returns 0 if there are no code-streams currently in use.
