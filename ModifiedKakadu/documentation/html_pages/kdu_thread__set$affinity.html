kdu_thread::set_cpu_affinity

bool set_cpu_affinity( kdu_long affinity_mask)

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

If supported, this function requests the scheduler to run the thread only on one of the CPU's whose bit is set in the supplied cpu_mask. CPU n is included if bit n is set in affinity_mask. The function returns true if the request is accepted. Although this function should be executable from any thread in the system, experience shows that some platforms do not behave correctly unless the function is invoked from within the thread whose processor affinity is being modified.

Return Value

False if the thread has not been successfully created, or the affinity_mask is invalid, or the calling thread does not have permission to set the CPU affinity of another thread, or operating support has not yet been extended to offer this specific feature.

Arguments

affinity_mask [kdu_long]
