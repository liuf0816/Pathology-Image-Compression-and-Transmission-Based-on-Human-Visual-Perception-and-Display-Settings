kdu_image_dims::set_resolution

void set_resolution( int ref_width, int ref_height, bool units_known, double xpels_per_metre, double ypels_per_metre)

[Declared in "../apps/image/kdu_image.h"]

Go to class description.

Synopsis

Used to set physical display resolution information, if known. Both the X and Y resolution values must be strictly positive. If units_known is false, only the ratio between these quantities has any meaning. In any case, the X and Y resolution values are expressed relative to an image resolution whose dimensions are given by ref_width and ref_height; these resolutions are scaled to match the actual dimensions of any individual image component.

Arguments

ref_width [int]

ref_height [int]

units_known [bool]

xpels_per_metre [double]

ypels_per_metre [double]
