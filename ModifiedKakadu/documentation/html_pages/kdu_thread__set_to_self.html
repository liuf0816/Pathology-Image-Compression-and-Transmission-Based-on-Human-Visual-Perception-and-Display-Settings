kdu_thread::set_to_self

bool set_to_self( )

[Declared in "../coresys/common/kdu_elementary.h"]

Go to class description.

Synopsis

You can use this function to create a reference to the caller's own thread. The resulting object can be passed to equals, get_priority, set_priority and set_cpu_affinity. Do not invoke the destroy function on a kdu_thread object created in this way.

Return Value

False if thread functionality is not supported on the current platform. This problem can usually be resolved by creating the appropriate definitions in "kdu_elementary.h".
