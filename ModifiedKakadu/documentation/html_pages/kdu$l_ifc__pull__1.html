Synopsis

Fills out the supplied line buffer's sample values before returning.

Arguments

line [kdu_line_buf &]

env [kdu_thread_env *]

If the object was constructed for multi-threaded processing (see the constructors for kdu_synthesis and kdu_decoder), you MUST pass a non-NULL env argument in here, identifying the thread which is performing the pull call. Otherwise, the env argument should be ignored.
