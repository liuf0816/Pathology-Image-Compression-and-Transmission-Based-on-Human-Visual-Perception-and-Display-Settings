Synopsis

This function searches through all compositing layers which are involved in the current presentation to find the top-most compositing layer in which point resides. If one exists, the function returns true, and uses layer_idx and codestream_idx to return information about the matching compositing layer and its primary codestream. Otherwise, the function returns false.

Arguments

point [kdu_coords]

Location of a point on the compositing grid, expressed using the same coordinate system as that associated with set_buffer_surface.

layer_idx [int &]

Used to return the zero-based index of the top-most compositing layer which contributes to imagery at location point. If the composited image consists only of a single codestream, the value of this argument is set to -1, while the value of codestream_idx is set to a non-trivial value. The value of this argument is not altered if the function returns false.

codestream_idx [int &]

Used to return the zero-based index of the primary codestream which contributes to imagery at point. The value of this argument is not altered if the function returns false.
